# Interactive Developer Dashboard (Static)

This is a self-contained dashboard-style portfolio:

- Timeline portfolio
- Storytelling portfolio
- Game: "Guess my hobby"
- Project showcase gallery
- Skills dashboard (radar + meters)
- Theme preference (auto/dark/light, persisted)

## Run

Open `index.html` in a browser.

If you prefer a local server (recommended for modules), from this folder run:

```powershell
python -m http.server 5173
```

Then visit `http://localhost:5173`.

## Edit content

Update the data in `assets/data.js`:

- `profile`
- `timeline`
- `story`
- `hobbies`
- `projects`
- `skills`
